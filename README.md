# Application Java – Gestion des étudiants 🎓
Application desktop développée avec Java et Java Swing permettant de gérer les informations des étudiants à travers une interface graphique simple et organisée.
## 🚀 Technologies utilisées
- Java
- Java Swing
- Gestion des événements
- Interface graphique desktop
## 🎯 Objectif du projet
Développer une application de gestion des étudiants permettant d’organiser les données liées aux étudiants, aux notes, aux matières et aux moyennes.
## 🖥️ Interfaces de l’application
L’application contient une interface d’accueil qui donne accès à plusieurs interfaces :
- Interface Étudiant
- Interface Notes
- Interface Moyenne
- Interface Matières
## ⚙️ Fonctionnalités principales
Chaque interface permet d’effectuer les opérations de base suivantes :
- Ajouter des informations
- Modifier des informations
- Supprimer des informations
- Afficher les données
- Gérer les actions à travers des boutons et des événements Java Swing
## 📌 Description
L’interface d’accueil permet de naviguer facilement entre les différentes parties de l’application.  
Les boutons permettent à l’utilisateur d’ajouter, modifier, supprimer et consulter les données liées aux étudiants, aux notes, aux matières et aux moyennes.
